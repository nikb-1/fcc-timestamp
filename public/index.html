Timestamp Microservice API


This API is a Free Code Camp - Back End Development Certification project.

It can be used by passing either a UNIX timestamp or a natural language date to the root end point as a parameter, i.e. "/:date_goes_here"

The API returns a JSON string with a "unix" property with a unix timestamp value and a "natural" property with a value representing a natural language date. Both values being representations of the passed parameter.

Incorrectly formatted parameters will return null values for the JSON objects properties.


Usage

https://fcc-timestamp-nikburazor.herokuapp.com/December%2015,%202015

https://fcc-timestamp-nikburazor.herokuapp.com/1450137600000

Output

{"unix":1450137600000,"natural":"December 15, 2015"}
